# Contenido y plantillas de carga

Esta carpeta se usara para organizar plantillas de carga de contenido antes de importarlas a la plataforma.

## Advertencia

No subir bancos de preguntas oficiales o informacion sensible sin revision previa. Para pruebas se deben usar datos de ejemplo o anonimizados.

## Plantillas previstas

- `plantilla_carga_contenido.csv`: estructura minima para preguntas.
- Plantilla Excel: se conserva como artefacto externo hasta definir si se subira al repositorio como archivo binario.
