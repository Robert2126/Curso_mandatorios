# Identidad visual

## Nombre recomendado

Aula Mandatorios.

## Subtitulo

Preparacion personal, practica y autoevaluacion.

## Concepto visual

Ruta de aprendizaje personal para avanzar por ciclos, cursos, temas, practicas y simulacros.

## Paleta sugerida

| Uso | HEX |
|---|---|
| Verde principal | #0B3D2E |
| Verde secundario | #1F7A4D |
| Verde claro | #E8F3ED |
| Dorado suave | #C9A646 |
| Fondo general | #F4F6F8 |
| Texto principal | #1F2937 |
| Texto secundario | #6B7280 |
| Error | #B42318 |
| Informacion | #2563EB |

## Tipografia

- Principal: Inter.
- Alternativa: Roboto.

## Estilo

- Educativo institucional moderno.
- Sobrio, claro y responsive.
- Gamificacion moderada.
- Iconos lineales y simples.

## Componentes visuales

- Boton primario: fondo verde principal, texto blanco.
- Boton secundario: fondo blanco, borde verde, texto verde.
- Tarjetas: fondo blanco, borde gris claro, sombra suave.
- Insignias: acento dorado suave.
- Barras de progreso: verde secundario.

## Advertencia de uso de identidad

No usar escudos o emblemas oficiales sin autorizacion. Para el prototipo se recomienda crear un emblema generico relacionado con aprendizaje, ruta y preparacion.
